# Example of Frontity site

## Getting started

```bash
$ git clone git@github.com:getshifter/example-frontity-wordpress.git
$ cd example-frontity-wordpress
$ npm install
$ npm run dev

SERVER STARTED -- Listening @ http://localhost:3000
  - mode: development
  - target: module

```

## Build

```bash
$ npm run build
```


## Deployment (Vercel (now.sh))

```bash
$ npx now login
$ npx now --prod
```
